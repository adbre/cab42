INSTALL
=======

Building the latest version on Windows
-----------------

Note: Make sure msysgit is installed.
It can be obtained from http://git-scm.com

    $ git clone https://adambrengesjo@bitbucket.org/adambrengesjo/cab42.git
    $ ./build.sh
    $ ./install.sh
    
Updating from git

    $ git pull
    $ ./build.sh
    $ ./install.sh
